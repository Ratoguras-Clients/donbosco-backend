# CNI Organization Website

## Project Overview

The **CNI Organization Website** is a comprehensive platform designed to enhance the digital presence and reach of the CNI organization. The website serves as a central hub for information, engagement, and updates regarding the organization’s activities, events, and sister organizations.

This project aims to provide an intuitive, responsive, and user-friendly experience for visitors while showcasing the organization’s mission, programs, and affiliated entities.

---

## Features

The website includes the following pages and functionalities:

### Main Pages

-   **Home:** Introduction and overview of CNI Organization.
-   **About Us:** Information about the organization’s mission, vision, and team.
-   **Contact:** Contact form and details for inquiries.
-   **Media:** Gallery of photos, videos, and multimedia content.
-   **Blogs:** Articles, news, and updates related to CNI activities.
-   **Events:** Upcoming and past events organized by CNI.
-   **Gallery:** Visual representation of events, achievements, and activities.
-   **Sister Organizations:** List of affiliated organizations with their respective pages, including a detailed **About Us** for each, along with a timeline of milestones and achievements.

### Additional Features

-   Responsive design for desktop and mobile devices.
-   Organized timeline for sister organizations’ achievements.
-   Dynamic media gallery for images and videos.
-   SEO-friendly structure for better online reach.
-   Easy-to-navigate menus and content sections.

## How to Run the Project

1. Clone the repository:

    ```bash
    git clone https://github.com/username/cni-organization-website.git
    ```

2. Navigate to the project folder:

    ```bash
    cd cni-organization-website
    ```

3. Install dependencies (if applicable):

    ```bash
    composer install   # For Laravel backend
    npm install        # For frontend dependencies
    ```

4. Set up your environment variables:

    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

5. Run the project:

    ```bash
    php artisan serve
    ```

6. Access the website in your browser at `http://localhost:8000`.
